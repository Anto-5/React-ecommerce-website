# React E-Commerce Store

A simple e-commerce website built using React. This project uses the DummyJSON API to display products, view product details, and manage a shopping cart.

## Features

- Product listing
- Product details page
- Add to cart
- Increase/decrease quantity
- Remove items from cart
- Bill summary

## Technologies Used

- React
- React Router DOM
- Context API
- CSS
- DummyJSON API

## Run Locally

```bash
npm install
npm run dev
```
